# Coze

> 字节跳动出品的低代码 Bot 平台——面向非技术用户，通过可视化配置快速搭建和发布 AI Bot。

| 属性 | 值 |
|------|-----|
| 厂商 | 字节跳动 |
| 语言 | - |
| 开源 | 否 |
| 官网 | [coze.com](https://www.coze.com/)[^coze-docs] |

## 技术亮点

- **插件生态**：内置丰富的第三方插件（搜索、绘图、数据查询等），Bot 通过配置而非编码接入外部能力。插件的发现和调用对用户透明
- **多渠道发布**：构建的 Bot 可一键发布到飞书、Discord、Telegram、微信等即时通讯平台，覆盖字节生态和主流社交平台
- **工作流画布**：与 Dify 类似的可视化流程编排，但更侧重对话场景的流程设计（对话分支、意图识别、多轮追问）
